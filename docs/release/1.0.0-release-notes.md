# CatchIt v1.0.0

First public release of CatchIt.

## Highlights
- Save selected text from any webpage with a quick popover
- Add tags and manage saved items in the dashboard
- Search, copy, open source links, and delete highlights
- Sync highlights to Notion with Auto Sync or Sync Now
- Test Notion connection and open the target database directly
- Default Side Panel entry experience
