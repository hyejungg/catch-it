# CatchIt 개인정보처리방침 템플릿

> 아래 내용은 템플릿입니다. 실제 운영자 정보/연락처/시행일을 채워서 공개 URL로 배포하세요.

## 1. 수집하는 정보
- 사용자가 확장에 입력한 Notion Integration Token
- 사용자가 입력한 Notion Database ID
- 사용자가 저장한 하이라이트 데이터(텍스트, URL, 제목, 태그, 저장 시각)

## 2. 정보 사용 목적
- 사용자가 요청한 웹 텍스트 저장 기능 제공
- 사용자가 요청한 Notion 동기화 기능 제공

## 3. 정보 저장 및 전송
- 하이라이트/설정 데이터는 사용자의 브라우저 `chrome.storage.local`에 저장됩니다.
- Notion 동기화 기능 사용 시에만 `https://api.notion.com`으로 데이터가 전송됩니다.

## 4. 제3자 제공
- 사용자 요청에 따른 Notion API 전송 외 별도 제3자 제공을 하지 않습니다.

## 5. 보관 및 삭제
- 사용자는 확장 내 삭제 기능으로 하이라이트를 삭제할 수 있습니다.
- 확장 제거 시 브라우저 로컬 저장 데이터는 브라우저 정책에 따라 삭제될 수 있습니다.
